# Medstrack User Management

Medstrack es un backend desarrollado en Java con Spring Boot, diseñado como un sistema de gestión y registro de usuarios con estándares modernos de seguridad, validación y escalabilidad. Su objetivo es servir como la base de una plataforma medtech enfocada en el seguimiento de medicamentos, recordatorios y funcionalidades orientadas a la salud.

Actualmente, el backend se encuentra en desarrollo, ampliando sus capacidades hasta convertirse en un microservicio robusto para:
- Registro de usuarios profesional (signup)  
- Inicio de sesión seguro (login)  
- Manejo de tokens y sesiones  
- Integración futura con otros servicios Medstrack  

Medstrack Backend está pensado tanto como un proyecto real medtech como un portafolio profesional para demostrar habilidades sólidas como Java Backend Developer.
